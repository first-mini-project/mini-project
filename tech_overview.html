⭐ 주요 구현 기능
  • 🎨
    웹 그림판 (Canvas API)
    브라우저에서 직접 그리기 · 지우기 · 색상 팔레트 · 실행 취소(Ctrl+Z) · 그림 저장. 기존 그림 불러와서 수정 가능.
  • 📚
    동화책 뷰어 (스토리북)
    페이지 넘기기, 장면별 배경 이미지, 그림 플로팅 애니메이션, TTS 재생 버튼. 기승전결 4단락 구조로 AI가 자동 생성.
  • 🎈
    물리 기반 플로팅 애니메이션
    그림 모음장에서 카드가 중력·반발력·마우스 인력으로 실시간 움직임. requestAnimationFrame 기반, 모바일에서는 자동으로 그리드 모드 전환 (배터리 절약).
  • 🔊
    멀티보이스 TTS
    내레이션: 여성 목소리(ko-KR-SunHiNeural) / 캐릭터 대화("...") : 남성 목소리(ko-KR-InJoonNeural). MP3 세그먼트 연결 방식으로 ffmpeg 없이 구현.
  • 🌐
    자동 폴백 구조 (이중 안전망)
    동화 생성: Gemini → Claude / 배경 이미지: 로컬 GPU → Pollinations.ai / TTS: Edge TTS → gTTS. 어느 API가 실패해도 서비스 중단 없음.
  • 🤖
    한국어 조사 자동 교정
    AI 생성 텍스트의 은/는, 이/가, 을/를, 으로/로 등 조사 오류를 유니코드 받침 분석으로 자동 수정. 정규식 + 한글 유니코드 공식 활용.
  • 🎮
    로딩 중 풍선 미니게임
    동화 생성 대기(약 15~30초) 동안 14종 풍선 터트리기 게임 자동 실행. 가중치 랜덤 등장, 크기·속도 무작위, 점수 시스템. MutationObserver로 로딩 상태 감지.
